# Scoring And Compliance Model

## Scoring Categories

The first scoring model uses five weighted categories:

- Hazard perception: 30%
- Road-rule compliance: 25%
- Vehicle control smoothness: 20%
- Emergency response: 15%
- Fuel efficiency and mechanical sympathy: 10%

Weights can be tuned per lesson type, route, licence class, and instructor policy.

## Event Types

- Observation: mirror checks, head checks, scanning intervals, blind-spot checks.
- Control: acceleration, braking, steering smoothness, gear selection, clutch handling.
- Compliance: speed, stop lines, signals, signs, lane use, roadblock instructions.
- Hazard: time-to-collision, near miss, unsafe following, pedestrian conflict, animal conflict.
- Efficiency: excessive revving, harsh acceleration, unnecessary idling, poor gear choice.
- Intervention: instructor pause, scenario injection, emergency stop, override.

## VID Preparation Traceability

The assessment system should support preparation for practical driving-test skills:

- hill start
- three-point turn
- parallel parking
- reversing/drums
- urban driving
- signs and rules
- pedestrian and cyclist awareness
- vehicle control

Production certification alignment must be validated with current VID requirements before being marketed as an official assessment tool.

## Road Standards Traceability

Road signs, markings, temporary roadworks controls, and signal behaviour should be traceable to the selected SADC-aligned road traffic sign references and Zimbabwean implementation decisions.

Each sign asset should store:

- sign code
- sign category
- source standard
- Zimbabwe text variant
- asset version
- approval status

## Audit Log Requirements

Each training session should record:

- learner ID
- instructor ID
- training centre ID
- date/time and offline sync status
- vehicle profile
- route and scenario pack versions
- headset/device profile
- score summary
- raw event timeline
- instructor interventions
- generated feedback

## Failure Conditions

Immediate critical events should be configurable by licence class and instructor policy. Examples:

- collision with pedestrian, cyclist, livestock, vehicle, or roadblock officer
- running a red light or stop control
- unsafe overtaking into oncoming traffic
- failure to stop for police roadblock
- loss of vehicle control
- exceeding speed threshold in school zone

## Feedback Style

Learner feedback should be specific and replay-linked:

- what happened
- where it happened
- why it mattered
- what to do next time

Avoid vague feedback such as "drive better" or "be careful".
